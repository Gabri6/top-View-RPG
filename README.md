"# top-View-RPG" 

A 2D top view RPG
controls :
    - arrows or z,q,s,d : Move your character (in black and red)
    - left shift : sprint
    - "space" : attack in front of you (to where the red square is)
    - "B" : launch a powerful attack all around you
    - "return" : restart the game once its game over
